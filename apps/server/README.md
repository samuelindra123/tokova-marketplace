# Multivendor Marketplace Backend

## Environment Variables

Copy `.env.example` to `.env` and fill in your values.

## Development

```bash
npm run start:dev
```

## API Documentation

Visit `http://localhost:3000/api/docs` for Scalar API documentation.

## Database

```bash
# Generate migrations
npm run db:generate

# Run migrations
npm run db:migrate

# Open Drizzle Studio
npm run db:studio
```
